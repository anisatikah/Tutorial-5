Solutions
PartPhraseAnswer

QUESTION 8c–e · CLO 2

Variance, Std Dev & Skewness — Baby Ages

Given Data (from previous slide)
Σf = 24
Σfx = 338
Σfx² = 5456
Formula & Solution
(c) Variance s² = Σfx²/(n−1) − (Σfx)²/[n(n−1)]
= 5456/23 − (338)²/(24×23)
= 237.217 − 206.964 = 30.2536
(d) Standard Deviation s = √variance = √30.2536 = 5.5
(e) Mean = 338/24 = 14.0833
Mode: modal class 12–17, L=12, d₁=11−6=5, d₂=11−2=9, c=5
Mode = 12 + (5/14)×5 = 13.7857
Pearson's Sk = (Mean − Mode)/s = (14.0833 − 13.7857)/5.5
Sk = 0.0541 → Positively skewed (right)
FINAL ANSWERS
s² = 30.25  |  s = 5.50  |  Sk = +0.054 → Right Skew
💡 Key Insight
Pearson's Coefficient: Sk > 0 → positively skewed (tail to the right). Sk < 0 → negatively skewed. Sk = 0 → symmetric.

QUESTION 9 · CLO 2

Mean, Std Deviation & Coefficient of Variation

Question
In a mid-semester exam, Aishah's total marks for all 10 subjects: ΣM = 650, and ΣM² = 43000. Find (a) mean and standard deviation, (b) coefficient of variation.
Given Data
n = 10
ΣM = 650
ΣM² = 43000
Solution
1
Mean = ΣM / n = 650 / 10 = 65
2
Variance s² = ΣM²/(n−1) − (ΣM)²/[n(n−1)]
= 43000/9 − (650)²/(10×9)
= 4777.78 − 4694.44 = 83.3334
3
Standard deviation s = √83.3334 = 9.1287
4
Coefficient of Variation = (s / mean) × 100%
= (9.1287 / 65) × 100% = 14.0442%
CV = (Standard Deviation / Mean) × 100%
FINAL ANSWERS
Mean = 65  |  s = 9.13  |  CV = 14.04%
💡 Key Insight
CV compares variability relative to the mean. A lower CV means more consistent data. Aishah's marks have moderate variation (~14%).

REFERENCE SHEET

All Key Formulas — Data Description

Mean (Grouped)

x̄ = Σfx / Σf

Mode (Grouped)

Mode = L + [d₁/(d₁+d₂)] × c
d₁ = f_mode − f_before
d₂ = f_mode − f_after

Median (Grouped)

Q₂ = L₂ + (c₂/f₂)(N/2 − Σf_B)

Quartile / Percentile

Q₁ = L + (c/f)(N/4 − Σf_B)
P_k = L + (c/f)(kN/100 − Σf_B)

Variance (Sample)

s² = Σfx²/(n−1) − (Σfx)²/[n(n−1)]

Standard Deviation

s = √(Variance)

Coefficient of Variation

CV = (s / x̄) × 100%

Pearson's Skewness

Sk = (Mean − Mode) / s
Sk > 0 → Right skew
Sk < 0 → Left skew

Relative Frequency

RF = f / Σf

Class Boundaries

Lower Boundary = LL − 0.5
Upper Boundary = UL + 0.5

Class Mark (Midpoint)

x = (LL + UL) / 2

Skewness by Comparison

Mode < Median < Mean → Right
Mean < Median < Mode → Left
Mean = Median = Mode → Symmetric
